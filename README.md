# Teste técnico Pague Bem Brasil

O projeto tem como objetivo a criação de uma landing page para uma empresa fictícia

Como iniciar o projeto

 - Garanta que você tenha o [Play Framework](https://www.playframework.com/) instalado em sua máquina
 - Clone o repositório utilizando o terminal com o camando: 
	 - `git clone https://github.com/AlipioGSF/paguebembrasil.git`
	 - Navague para o diretório do projeto com `cd .\paguebembrasil\`
 - Inicie o projeto com o comando `sbt run` no terminal
 - Abra uma aba no seu navegador em localhost:9000
